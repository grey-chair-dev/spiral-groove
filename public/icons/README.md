# PWA Icons

This directory contains Progressive Web App (PWA) icons for Spiral Groove Records.

## Required Icons

Please add the following icon files:

- `icon-192x192.png` - 192x192 pixels, PNG format
- `icon-512x512.png` - 512x512 pixels, PNG format

## Icon Specifications

- **Format**: PNG
- **Sizes**: 
  - 192x192px (for Android home screen)
  - 512x512px (for splash screens and high-res displays)
- **Purpose**: Both icons should be "any maskable" (work with adaptive icons)
- **Design**: Should represent Spiral Groove Records brand (vinyl record, spiral groove, etc.)

## Usage

These icons are referenced in `/public/manifest.json` and will be used when users install the PWA to their home screen.

## Generation Tools

You can create these icons using:
- Design tools (Figma, Adobe Illustrator, etc.)
- Online PWA icon generators
- Image editing software (Photoshop, GIMP, etc.)

Make sure the icons are optimized for web (compressed but high quality).
