# Placeholder Images

This directory contains placeholder images for different product categories.

## Required Images

Please add the following placeholder images:

- `vinyl.jpg` - For vinyl records and music-related products
- `equipment.jpg` - For turntables, amplifiers, and audio equipment
- `crafts.jpg` - For vinyl crafts and handmade items

## Image Specifications

- Format: JPG
- Recommended size: 1200x1200px minimum
- Aspect ratio: 1:1 (square) for product images
- File size: Optimized for web (< 200KB each)

## Usage

These images are automatically used by the placeholder system when:
- Square API is not configured (demo mode)
- Product images are not available
- Category-specific placeholders are needed

The system will automatically select the appropriate placeholder based on product category:
- Vinyl records → `vinyl.jpg`
- Turntables, amplifiers, accessories → `equipment.jpg`
- Crafts and handmade items → `crafts.jpg`

## Alt Text

All placeholder images use descriptive alt text following the format:
- `"[Category] at Spiral Groove Records"`
- Example: `"Vinyl Records at Spiral Groove Records"`
