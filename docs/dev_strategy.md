4. UX/UI Design Approach
🎨 Visual Design
Style: Retro-modern minimalism — inspired by analog warmth and clean layouts (e.g., Plaid Room aesthetic with improved clarity).

Color Palette: Deep black, cream, muted teal, amber accents.

Typography:

Headings: Playfair Display (vintage character)

Body: Inter / Poppins (modern legibility)

Imagery: Lifestyle photography, vinyl close-ups, community events, and nostalgic textures.

🧠 UX Principles
Mobile-First: 70%+ users on mobile — vertical scrolling with sticky “Shop” button.

Clarity: Clear hierarchy; no clutter.

Conversion-Focused:

“Shop Now” or “Reserve for Pickup” on all product cards.

“Join the Groove” pop-up with 10% first order discount.

Persistent cart icon in header.

Accessibility:

Contrast ratio ≥ 4.5:1

ARIA tags for nav and product grids

Skip-to-content link for screen readers
